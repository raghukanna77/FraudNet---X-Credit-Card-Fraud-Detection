🎮 DEMO MODE EXPLAINED:
───────────────────────────────────────────────────────────────────────────

How Demo Mode Works:

Instead of trained ML models, the system uses intelligent heuristics:

1. Amount Risk (40% weight)
   → Higher amounts = higher risk
   → Normalized to 0-1 scale

2. V Feature Extremeness (40% weight)
   → PCA features far from 0 = anomalies
   → Maximum deviation indicates outliers

3. Mean V Value (20% weight)
   → Overall feature pattern analysis
   → Consistent high values = suspicious

Result: Realistic fraud scores without requiring:
   ✗ Training dataset (138MB CSV)
   ✗ Model training (hours of compute)
   ✗ Heavy ML libraries (GBs of disk space)

The heuristic approach provides:
   ✓ Instant predictions
   ✓ Explainable results
   ✓ Realistic risk scores
   ✓ Working demonstration

🎓 UNDERSTANDING THE RESULTS:
───────────────────────────────────────────────────────────────────────────

Risk Levels:
   🟢 LOW (0-30):       Normal transaction, approve
   🟡 MEDIUM (30-60):   Review recommended
   🟠 HIGH (60-85):     Likely fraud, block and investigate
   🔴 CRITICAL (85-100): Almost certain fraud, block immediately

Fraud Probability:
   → Percentage chance of fraud (0-100%)
   → Based on heuristic analysis in demo mode

Confidence:
   → How certain the system is (0-100%)
   → Higher = more reliable prediction

Feature Importance:
